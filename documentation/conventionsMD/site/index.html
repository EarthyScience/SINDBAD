  • About SINDBAD

This documentation describes the key concepts and then provides the conventions and guidelines for development of the framework for Strategies to INtegrate Data and BiogeochemicAl moDels (SINDBAD).

The SINDBAD framework provides a seamless integration of model and multiple data streams using different optimization schemes.

Developers

SINDBAD is being developed at the Department of Biogeochemical Integration of the Max Planck Institute for Biogeochemistry in Jena, Germany. Following is the list of SINDBAD developers.
